--- layout: post title: Minify JS, CSS files on Save in Sublime Text date: '2014-06-16T15:19:00.000+03:00' author: Fred Muya tags: modified_time: '2014-06-16T15:19:48.144+03:00' blogger_id: tag:blogger.com,1999:blog-3630098189866392828.post-8404432135741283494 blogger_orig_url: http://encore254.blogspot.com/2014/06/minify-js-css-files-on-save-in-sublime.html --- We all know the advantages of using minified files in your web apps, i.e. faster loading, smaller payload, etc, it would make sense if all editors that we use have the feature built in, so that we don't have to worry about it.

I use Sublime Text for most of my work, and much as there are plugins to accomplish the minification of JS & CSS files, an essential feature that I haven't come across is minification on Save.

To solve this, I created a simple plugin to accomplish just this. It requires you to have the Sublime-Minifier extension, here: https://github.com/bistory/Sublime-Minifier

To start using it: 

  • Grab this Gist: https://gist.github.com/muya/1cd6ce82490d2c0f2e0a
  • In Sublime-Text, click on Tools->New Plugin...
  • Replace all the code in the new file with content from the Gist, then Save

Now, when you save any of your JS & CSS files, a corresponding file (.min.js/.min.css) will be created/updated in the directory.

Enjoy!
